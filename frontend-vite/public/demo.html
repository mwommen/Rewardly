Rewardly demo

Catch the Amex Platinum Lululemon benefit at checkout.

This launcher is the local demo path for the Chrome extension MVP. Use it to rehearse setup, open the checkout page, and record the core story.

Demo Flow

  1. 1
    Confirm backend and frontend are running. Backend on localhost:5001. Frontend on the current Vite port.
  2. 2
    Reload the unpacked extension. In chrome://extensions, reload Rewardly after code changes.
  3. 3
    Open the checkout page. The extension should detect Lululemon and show the Amex Platinum benefit.
  4. 4
    Click Enroll. Amex should route to the specific logged-in Lululemon benefit flow.
